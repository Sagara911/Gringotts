# Nobi 采集与翻译扩展

在任意网页右键图片，可以选择「保存到 Nobi」，图片连同来源页面 URL 一起存入素材库。

选中网页上的英文或其他文本后右键，可以选择「Nobi 翻译」。扩展会调用正在运行的 Nobi 本地接口翻译，并在当前网页右下角显示结果。

图片保存在 `图片\Nobi\` 文件夹，文件透明可见，不锁定。

## 安装（Chrome / Edge）

1. 打开 `chrome://extensions/`（Edge 是 `edge://extensions/`）
2. 右上角打开开发者模式
3. 点「加载已解压的扩展程序」，选择本文件夹（`browser-extension/`）
4. 完成。任意网页右键图片或选中文字右键即可看到 Nobi 选项。

## 使用前提

- Nobi 应用必须在运行，它内置了本地接收服务 `127.0.0.1:21420`
- 图片采集成功后应用会自动刷新并显示新素材，来源列会记录页面 URL
- 翻译优先使用 Nobi 设置里的 OpenAI 兼容模型；没有可用模型时会走内置离线术语兜底

## 常见问题

- 提示“保存失败 / 翻译失败 / Nobi 是否在运行？”：先打开 Nobi 应用再试
- 个别图片存不下来：少数网站对图片有防盗链保护，扩展拿不到原图数据
- 个别页面不显示翻译浮层：浏览器内置页面或权限受限页面不允许注入，扩展会改用浏览器通知显示结果
